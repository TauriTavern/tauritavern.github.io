## 下载建议

- `Windows`：优先使用安装版；便携版适合免安装或放在移动磁盘里使用。
- `macOS`：如果当前只看到 Apple Silicon，说明最新版本暂时只提供 ARM 构建。
- `Linux`：优先使用 `AppImage`；若你依赖系统包管理器，再选 `DEB` 或 `RPM`。
- `Android`：大多数手机和平板都应该选择 `ARM64`。
- `iOS`：`IPA` 需要通过 iOS 支持的签名或侧载方式安装。
