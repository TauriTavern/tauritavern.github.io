## Recommendations

- `Windows`: start with the installer; portable is best for no-install setups.
- `macOS`: if only Apple Silicon appears, the latest release currently ships ARM builds only.
- `Linux`: `AppImage` is the default choice; use `DEB` or `RPM` when you prefer your distro package manager.
- `Android`: most phones and tablets should use `ARM64`.
- `iOS`: `IPA` packages need to be installed through a supported iOS signing or sideload workflow.
